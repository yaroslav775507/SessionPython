# Документация проекта

## 1. Назначение проекта
В ноутбуке реализован полный пайплайн для задачи классификации изображений (папка /Weather Resized).
Проект реализует полный процесс решения задачи классификации изображений с использованием пользовательского датасета. 
В ходе работы формируется кастомный датасет, создаётся и обучается модель, а также проводится итоговая оценка качества.

---

## 2. Структура работы

### 2.1. Подготовка данных

* Определён кастомный класс датасета, отвечающий за загрузку изображений и сопоставление их с метками.
* Реализовано применение преобразований и подготовка данных для загрузчиков (train/val/test).

### 2.2. Определение модели

* Реализована архитектура модели для классификации изображений.
* Выполнена инициализация параметров и подготовка к обучению.

### 2.3. Обучение модели

* Настроен тренировочный цикл: прямой проход, расчёт функции потерь, обратное распространение ошибки и обновление весов.
* Присутствует логирование ключевых метрик на протяжении обучения.

### 2.4. Оценка

* После обучения производится расчёт итоговых метрик на валидационных или тестовых данных.
* Строится и анализируется confusion matrix для оценки качества предсказаний по каждому классу.

---

## 3. Основной функционал

### Dataset

* Загрузка изображений из пользовательских директорий.
* Приведение данных к необходимому формату.
* Возврат изображения и соответствующей метки.

### Model

* Архитектура нейронной сети для классификации.
* Готовность к обучению и инференсу.

### Training

* Оптимизация параметров модели.
* Подсчёт тренировочных метрик.

### Evaluation

* Подсчёт точности и других метрик.
* Матрица ошибок для анализа поведения модели.

---

## 4. Результат

Ноутбук обеспечивает полный цикл обучения модели классификации изображений и предоставляет удобный инструмент для анализа результатов.
